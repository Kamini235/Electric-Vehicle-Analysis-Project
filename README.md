# Electric-Vehicle-Analysis-Project

 Electric Vehicle Analysis Project 
 
Goal of the Dashboard is to deliver an interactive visual tool that: Enables users to explore Electric vehicle analysis globally. Supports decisions such as Future planning, regional marketing, or infrastructure investment. Uncovers trends in different terrain, Government subsidy policy, accessibility, and availability of Charging station in particular region.

Purpose

This project has been incredibly interesting! 
While building this dashboard I learned a lot of techniques in power bi and my confidence has doubled. 

EV Analytics Dashboard is a visually engaging and analytical Power BI report designed to help users explore and compare over different Makers and category of vehicle across different States of India. The dashboard focuses on highlighting major features like CAGR, Growth rate. This tool is intended for use by AtliQ Motor for their future expansion in India. and other electric vehicle manufacturing companies, and data-driven strategists who seek to understand trends and characteristics of EV sale globally.

 Tech Stack
 
List the key technologies used to build the dashboard.
Example: The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main data visualization platform used for report creation.
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
• 📝 Data Modeling – Relationships established among tables  ( dim date, state, Maker, and fact table ) to enable cross-filtering and aggregation.
• 📁 File Format – .pbix for development and Canva for making video.
